# Project Brief: rag4code

## Цель проекта
Создание системы семантического поиска по коду (RAG), которая позволяет AI-агентам и пользователям быстро находить релевантные фрагменты кода в больших репозиториях.

## Ключевые возможности
- Индексация локальных файлов кода.
- Генерация эмбеддингов (локально или через API).
- Векторный поиск (Qdrant/LanceDB).
- Интеграция с AI-агентами через MCP (Model Context Protocol).

## Целевая аудитория
Разработчики и AI-агенты, работающие со сложными кодовыми базами.
