# Project Brief: rag4code

## Цель проекта
Создание модульной системы семантического поиска по коду (RAG), построенной по принципу "LEGO", где каждый компонент является заменяемым модулем с четким интерфейсом.

## Ключевые возможности
- **Гибкая индексация**: Поддержка различных стратегий чанкирования (текущая: Simple Split).
- **Сменный векторный поиск**: Основной движок Qdrant с возможностью подключения других БД.
- **Локальные и удаленные эмбеддинги**: Использование локального Ollama (Qwen3-Embedding) или внешних API.
- **Многоканальный доступ**: Работа через Web UI, CLI и MCP (Model Context Protocol).

## Целевая аудитория
Разработчики и AI-агенты, работающие со сложными кодовыми базами.
